# FYP Project - Jewelry Recommendation System

This is the Final Year Project for a jewelry recommendation system based on face shape analysis.

## Project Structure

- **app/**: Main application directory
  - **model/**: Machine learning models
  - **assets/**: Dataset and jewelry assets
  - **saved_models/**: Trained model files
  - **uploads/**: User uploaded images
  - **outputs/**: Generated recommendations

## Getting Started

1. Install dependencies: `pip install -r requirements.txt`
2. Run the application: `python app/main.py`
